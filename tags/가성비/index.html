맥 미니 m1 깡통 두달반 사용기

8월말에 m1 맥 미니 기본 모델 (8GB/256GB)를 샀으니 이제 한 두 달 반 정도 됐습니다. 업무용 등으로 맥을 써 본 경험은 있었지만 개인용 시스템으로 풀타임? 사용한 적은 이번이 처음이라 비슷한 고민이 분들에게 참고가 될까하여 정리해봅니다. 왜 샀어요? RISC UNIX Workstation을 써 보고 싶었기 때문이죠. 다른 말로는 MS 오피스 돌아가는 유닉스 머신! 유닉스 머신이긴 해요? 넵. 유닉스 머신입니다…만, 터미널 환경에서만 그렇다고 생각하면 됩니다. 터미널에서는 tcsh 대신 ksh 쓰는 FreeBSD 느낌으로 대하면, 애플 제공 + homebrew 하면 있을?
